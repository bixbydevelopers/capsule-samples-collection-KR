## Overview

이 캡슐은 간단한 퀴즈 게임 캡슐입니다. 
해당 캡슐을 통하여 Continuation, enum 타입 활용, 외부 API 사용등에 대하여 알 수 있습니다.

## Example Flow 예시

```
퀴즈게임 할래
아무거나로 할래 (퀴즈 장르 선택 view)
참거짓 타입으로 (퀴즈 유형 선택 view)
첫 번째로 해줘 (문제 답안 선택 view)
```
