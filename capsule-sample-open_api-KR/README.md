## Open API를 활용한 캡슐 Repository

### 1. 위치기반 관광정보 조회 (example.trip)

이 캡슐은 국문 관광정보 서비스 OPEN API를 활용하여, 주변의 관광정보를 보여주는 서비스를 제공합니다.

### 2. 공적 마스크 판매정보 (example.masksales)

이 캡슐은 공적 마스크 판매정보 OPEN API를 활용하여, 주변의 공적 판매 약국을 보여주는 서비스를 제공합니다.
